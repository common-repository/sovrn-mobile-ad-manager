== Description ==
Get your WordPress site mobile-enabled and start monetizing those mobile readers with sovrn mobile ad manager!

WPtouch is a mobile plugin for WordPress that automatically enables a simple and elegant mobile theme for mobile visitors of your WordPress website.  The administration panel allows you to customize many aspects of its appearance, and deliver a fast, user-friendly and stylish version of your site to your mobile visitors, without modifying a single bit of code (or affecting) your regular desktop theme.  Requires WPTouch 3.x or higher.

sovrn mobile ad manager extends the functionality of the WPTouch plugin through the support of sovrn ads.  Ads will be viewed by readers of your site on mobile devices, empowering you to increase earnings from your site from a growing mobile audience.

Before you start, please visit www.sovrn.com to create an account and apply for mobile ad tags.

For WordPress blogs only, download the WPTouch plugin and the sovrn ad manager plugin to get started.


== Installation ==
1. Upload `sovrn-mobile-ad-manager` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
